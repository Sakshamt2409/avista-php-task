# Avista Core PHP Practical Task

Core PHP + MySQL assignment implementation with:

- User registration/login with validation and bcrypt password hashing
- AJAX product add, edit, delete, and list operations
- Multiple product image upload and AJAX image delete
- Dynamic product specification fields
- AJAX category, price range, and keyword filters
- Session-based cart with add, quantity update, remove, and total price

## Setup

1. Create/import the database using `database.sql` in phpMyAdmin or MySQL CLI.
2. Update database settings in `config/config.php` if your WAMP credentials differ.
3. Make sure `uploads/products` is writable by PHP.
4. Open `http://localhost/MyProject/avista-php-task/index.php`.

Default database name: `avista_php_task`.
